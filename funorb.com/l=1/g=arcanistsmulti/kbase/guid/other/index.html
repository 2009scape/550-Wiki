1. Wie kann ich eine technische Anfrage einreichen?

Bevor Sie uns eine technische Anfrage schicken, nehmen Sie sich bitte einen Moment Zeit, um sicherzustellen, dass ihr Computer auf dem neuestem Stand ist. Folgen Sie dafür den unten stehenden Anweisungen.

Halten Sie Windows auf dem neuesten Stand


Um Upgrades für das Windows-Betriebssystem herunterzuladen, besuchen Sie bitte das Windows Download Centre unter: http://windowsupdate.microsoft.com/


Halten Sie Java auf dem neuesten Stand


Die VM (Virtual Machine), die verwendet wird, um FunOrb-Spiele zu betreiben, muss gelegentlich aktualisiert werden, da neue Versionen entwickelt und veröffentlicht werden. Diese Updates findet man auf der Homepage von Sun Java, die unter der folgenden Adresse gefunden werden kann: java.com/en/download/index.jsp


Auch ihren Webbrowser (z. B. Internet Explorer, Firefox) sollten sie auf neuestem Stand halten.


Egal welchen Webbrowser sie verwenden, wir empfehlen Ihnen, die neueste Version zu verwenden. Damit sollte das Spiel ohne Probleme laufen und Ihre Internetsicherheit verbessert werden.

Um Ihren Webbrowser auf neuestem Stand zu halten, sollten Sie die Webseite des Herstellers besuchen oder den dafür vorgesehenen Link verwenden, den Sie mit Ihrem Browser erhalten haben.


Wenn Sie Ihren Computer auf den neuesten Stand gebracht haben, klicken Sie bitte HIER, um fortzufahren.


2. Wo finde ich meine Systeminformationen?

Wie Sie Ihre Systeminformationen für das Windows-Betriebssystem finden.


Um Ihre Sofware-Systeminformationen zu finden, sowie einige Informationen über Ihre Hardware, führen Sie bitte folgende Schritte durch:
  1. Startmenü > Einstellungen > Systemsteuerung
  2. Wählen Sie das System-Symbol aus
  3. Auf der Registerkarte 'Allgemein' sollten Sie Informationen darüber finden können, welche Version von Windows Sie verwenden, wie viel RAM Sie haben und manchmal auch die Geschwindigkeit Ihres Prozessors erfahren können.

Wie Sie herausfinden, welche Java-Version Sie verwenden


Um Ihre Java-Version zu identifizieren (nur, wenn Sie Sun Java installiert haben):
  1. Startmenü > Einstellungen > Systemsteuerung
  2. Wählen Sie das Java-Symbol aus
  3. Klicken Sie auf die Schaltfläche 'Info', um zu sehen, welche Java-Version Sie verwenden.

Wie Sie Ihre Systeminformationen für das Betriebssystem MacOS von Apple-Computern finden.
  1. Klicken Sie auf das Apple-Symbol in der oberen linken Ecke Ihres Bildschirms und wählen Sie 'Über diesen Mac'.
  2. In dem Fenster, dass ich jetzt öffnet, sehen Sie sowohl die Version Ihres Betriebssystems als auch Informationen über den Prozessor und den Speicher Ihres Mac.

Wie Sie Systeminformationen zu Ihrem Browser herausfinden
  1. Öffnen Sie Ihren Browser (Internet Explorer, Firefox usw.)
  2. Wählen Sie das Hilfe-Menü und wählen Sie 'Info' oder 'Über' (die genaue Bezeichnung kann sich von Browser zu Browser unterscheiden) aus. Damit sollten Sie erfahren, welche Browserversion Sie verwenden.

3. Kann man die Größe des FunOrb-Spielfensters verändern?

Falls Sie ein FunOrb-Mitglied sind, können Sie den Vollbild-Modus verwenden. Spielern der kostenlosen Version steht dieser Modus allerdings nicht zur Verfügung. Außerdem muss Ihr Computer die Mindestvoraussetzungen des jeweiligen Spiels erfüllen, die Sie auf der "Weitere Infos"-Seite zum Spiel finden.

Vollbild setzt die Benutzung des signierten Applets voraus. (Das signierte Applet ist voreingestellt. Wenn Sie es ändern möchten, müssen Sie 'Java-Optionen' unter 'Spiele' im Menü am oberen Seitenrand auswählen.) Wir versuchen, Vollbild auf so vielen Rechnern wie möglich verfügbar zu machen, allerdings gibt es immer Computer, auf denen es nicht funktionieren wird. Falls Sie Schwierigkeiten haben, sollten Sie Ihre Java-Version und Grafikkartentreiber aktualisieren.

4. Wie deaktiviere ich Suns Java-Plugin?

Wählen Sie aus dem Menü von Internet Explorer die Option Extras > Internetoptionen. In diesem Fenster klicken Sie auf den Reiter 'Erweitert' und suchen nach der Sektion Java (Sun). Hier können Sie das Plugin deaktivieren.

6. Das Spiel ist sehr langsam. Was ist los?

Falls Sie dieses FunOrb-Spiel zum ersten Mal ausprobieren oder das Spiel seit Ihrem letzten Zugriff darauf aktualisiert wurde, kann der Ladevorgang etwas länger dauern, da die Daten zuerst wieder auf Ihren Rechner geladen werden müssen. Falls das nicht der Fall ist:

Dieses Problem wird normalerweise dadurch verursacht, dass Sie FunOrb mit einem langsamen Computer spielen oder die Internetverbindung zwischen Ihrem Computer und unseren Servern schlecht ist. Selbst wenn Sie eine sehr schnelle Verbindung besitzen, kann es gut sein, dass ein Server auf dem Weg von Ihnen zu uns zu langsam arbeitet und dadurch das Problem verursacht.

Folgende Schritte könnten das Problem beheben oder reduzieren:

  • 1. Versuchen Sie, auf einem anderen Server zu spielen. Die Liste der Server finden Sie auf der Seite 'Weitere Infos' des Spiels über der Schaltfläche 'Beschreibung'.

  • 2. Gehen Sie sicher, dass kein anderes Programm Ihre Internetverbindung auslastet. Schließen Sie alle Programme, die Sie nicht aktiv verwenden. Sie sollten FunOrb nicht spielen, wenn Sie sich gerade Dateien aus dem Internet herunterladen oder Internet-TV ansehen, da solche Aktivitäten Ihre Internetverbindung stark beanspruchen.

  • 3. Spielen Sie zu einer anderen Tageszeit, wenn das Internet nicht so stark ausgelastet ist und daher die verschiedenen Verbindungen nicht so intensiv beansprucht werden.

  • 4. Im Menü 'Spiele' finden Sie einen Link zu den 'Java-Optionen'. Klicken Sie auf die Schriftrolle am unteren Rand und wählen Sie 'Digital signiertes Applet mit Suns Java-Version' aus und starten Sie dann das Spiel wie gewohnt.

  • 5. Falls Sie eine alte Version von Windows verwenden, gehen Sie sicher, dass der Java-'Just-In-Time'-Compiler aktiviert wurde. Im Internet Explorer unter: Extras > Internetoptionen > Erweitert > Microsoft-VM.

  • 6. Rüsten Sie Ihren Rechner auf.

  • 7. Verwenden Sie einen anderen Internetprovider.


Unsere Server verfügen über eine hervorragende Internetverbindung, sodass es sehr unwahrscheinlich ist, dass das Problem von uns verursacht wird.
