O jogo não é carregado

  1. Consigo ver a tela principal do jogo, mas só informa "Carregando...aguarde". O que devo fazer?
  2. Não consigo carregar um jogo do FunOrb. Nem mesmo a tela de título é carregada! O que devo fazer?
  3. Os jogos travam antes mesmo de serem carregados. O que devo fazer?
  4. O site está carregando muito lentamente. O que devo fazer?

1. Consigo ver a tela principal do jogo, mas só informa "Carregando...aguarde". O que devo fazer?

Se estiver usando um PC com 1 GB ou mais de RAM, pode ocorrer um conflito de memória. Esse problema é causado pelo Java, que não fornece ao jogo do FunOrb memória suficiente para trabalhar, e pode ser resolvido seguindo-se as etapas abaixo.

Observação: Só execute estas etapas se você tiver competência técnica ou se for o adulto responsável. A Jagex não assume responsabilidade por perda de dados resultante do uso desta correção.

- Abra o Painel de Controle do Windows
- Abra o Painel de Controle do Java
- Selecione a guia Java
- Na seção "Configurações de tempo de execução de applets Java" selecione "Exibir..." (não nas configurações de Aplicativos mas na de applets)

Sua versão de JRE deve estar listada. Na caixa identificada como Parâmetros de tempo de execução, inclua algo como:

-Xmx256M
Não esqueça o sinal de menos no início. Isso deve aumentar a memória máxima disponível para Java para 256 MB.
Não configure este valor para mais da metade da memória RAM máxima, do contrário o Windows terá problemas de execução.

Como os jogos do FunOrb foram projetados para ser executados no Windows usando o navegador Internet Explorer, só podemos oferecer suporte relativo a estes produtos. Caso esteja usando um sistema operacional ou navegador diferente, consulte o manual desses softwares para descobrir os procedimentos equivalentes.

Caso as dificuldades continuem, entre em contato novamente com a equipe do Suporte ao Cliente para obter mais ajuda. Lembre-se de incluir todas as alterações que ocorreram no desempenho do jogo após executar as ações acima, assim como as especificações do computador (velocidade da CPU, velocidade da conexão com a Internet, RAM e espaço livre no disco rígido).

2. Não consigo carregar um jogo do FunOrb. Nem mesmo a tela de título é carregada! O que devo fazer?

Tente as opções abaixo, na ordem apresentada, para resolver seu problema técnico. Se a primeira opção não solucionar o problema, consulte a opção seguinte.

  • Tente se conectar a um servidor diferente. A lista de servidores de jogos encontra-se na página 'Mais informações', logo acima do botão 'Descrição'. Se um de nossos servidores estiver com problemas técnicos, isso deverá resolver o problema até que nós o tenhamos consertado.

  • Seu navegador pode estar mantendo em cache uma versão antiga do jogo - tente limpar o cache da Web. Para isso, no Internet Explorer, na barra de opções, escolha Ferramentas > Opções da Internet > Excluir os arquivos de Internet temporários

  • Na caixa de seleção "Jogos" do site FunOrb, há um link para "Opções Java". Aqui você verá uma caixa de seleção: clique no menu e selecione Miniaplicativo assinado usando Java da Sun e depois tente carregar o jogo da forma usual. Com frequência, este procedimento corrige o problema.

  • Lembre-se de concordar com o pop-up de advertência de segurança que aparecerá alguns segundos após escolher o novo servidor de jogos. Caso o pop-up não apareça, reinicie o computador e clique em 'Sim' quando solicitado a confiar no miniaplicativo do FunOrb.

  • Na caixa de seleção "Jogos" do site FunOrb, há um link para "Opções Java". Aqui você verá uma caixa de seleção. Clique no menu e selecione Miniaplicativo não assinado usando Java da Sun e depois tente carregar o jogo da forma usual. Este procedimento corrigirá o problema caso você esteja jogando em um computador de uma escola ou de um café Internet em que o acesso ao computador seja limitado.

  • Faça download do Mozilla Firefox a partir de www.mozilla.org e tente usá-lo para jogar no FunOrb, em vez de usar o Internet Explorer.

  • Nossos servidores podem estar temporariamente desativados para manutenção ou atualização. Aguarde uma hora e tente novamente.

  • Assegure-se de que o computador tem capacidade suficiente para executar o jogo em questão. Cada jogo do FunOrb tem especificações mínimas individuais, que se encontram na página 'Mais informações' de cada jogo. Caso o computador esteja muito próximo dessa especificação, feche todos os programas que não sejam necessários antes de executar o jogo.


Se ainda houver problemas ao carregar o jogo, entre em contato com o Suporte Técnico.

3. Os jogos travam antes mesmo de serem carregados. O que devo fazer?

Se estiver tentando jogar no FunOrb em um computador com Windows Vista e Internet Explorer, talvez o Windows esteja configurado para executar em 'Modo Protegido'. Esse modo não permite que os jogos do FunOrb usem cache local e, por conseguinte, travam. Para resolver o problema, siga as etapas abaixo:

- No Internet Explorer, vá para Ferramentas > Opções da Internet > Segurança > Sites confiáveis > Sites - Desmarque a caixa intitulada 'Exigir verificação do servidor (https:) para todos os sites desta zona' - Na caixa 'Adicionar este site à zona', digite '*.funorb.com' (sem os apóstrofes) e clique em 'Adicionar'.

Como opção, faça download do navegador Mozilla a partir de www.mozilla.org e tente usá-lo para jogar no FunOrb, em vez de usar o Internet Explorer.



Se ainda houver problemas ao carregar o jogo, entre em contato com o Suporte Técnico.

4. O site está carregando muito lentamente. O que devo fazer?

Algumas vezes nosso site é carregado lentamente após uma atualização, porque todos tentam fazer download dos jogos mais atuais ao mesmo tempo. Aguarde alguns instantes e tente recarregar o site.
