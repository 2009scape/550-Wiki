1. Como posso enviar uma consulta técnica?

Antes de enviar um formulário do Suporte Técnico, reserve um momento para verificar se o computador está atualizado. Para isso, siga as instruções abaixo:

Mantenha o Windows atualizado


Para fazer download de atualizações do sistema operacional do Windows, visite a Central de Downloads do Windows em http://windowsupdate.microsoft.com/


Mantenha o Java atualizado


A VM (máquina virtual) usada para executar os jogos do FunOrb deverá ser atualizada à medida que novas versões forem desenvolvidas e liberadas. Essas atualizações estão localizadas na página da Sun Java, que pode ser encontrada no endereço a seguir: java.com/en/download/index.jsp


Mantenha o navegador da Web atualizado (por exemplo, Internet Explorer, Firefox)


Qualquer que seja o navegador da Web usado, recomendamos que seja a versão mais atualizada. Isso ajudará a executar os jogos mais facilmente e ajudará a manter a segurança na Internet.

Atualize o navegador da Web visitando o site oficial do desenvolvedor ou usando o link correto fornecido pelo navegador.


Quando o computador estiver atualizado, clique AQUI para continuar.


2. Como encontro as informações do meu sistema?

Como localizar as informações sobre sistema no sistema operacional do Windows


Para localizar as informações sobre o software no sistema, assim como algumas informações sobre o hardware, execute as etapas a seguir:
  1. Menu Iniciar > Configurações > Painel de Controle
  2. Selecione o ícone 'Sistema'
  3. Na guia 'Geral' aparecem informações sobre a versão do Windows em execução, a quantidade de memória RAMe, em algumas situações, a velocidade da CPU

Como localizar a versão do Java


Para localizar as informações do Java (somente se você tiver instalado o Sun Java):
  1. Menu Iniciar > Configurações > Painel de Controle
  2. Selecione o ícone 'Java'
  3. Clique no botão 'Sobre' para consultar a versão do Java em execução

Como localizar as informações sobre o sistema no Mac
  1. Clique no ícone azul da Apple no canto superior esquerdo da tela e selecione 'Sobre este Mac'.
  2. Na janela exibida, você verá a versão do sistema operacional, assim como as informações sobre o processador e a memória do Mac.

Como localizar as informações sobre oe sistema do navegador
  1. Abra o navegador (Internet Explorer, Firefox etc.)
  2. Selecione o menu 'Ajuda' e escolha 'Sobre'. Serão exibidas as informações da versão do navegador.

3. A janela dos jogos do FunOrb pode ser redimensionada?

Se for membro do FunOrb, sim, há um modo 'tela cheia'. Para não-membros, entretanto, não há modo de tela cheia.

O modo de tela cheia requer o uso do miniaplicativo assinado. O miniaplicativo assinado é o padrão. Caso você queira verificá-lo ou alterá-lo, selecione 'Opções Java' no menu 'Jogos', no alto desta página. Tentamos oferecer suporte ao modo de tela cheia em tantos computadores quanto possível, mas sempre há alguns computadores em que não é possível fazer funcionar. Caso tenha problemas, tente atualizar a versão do Java ou dos drivers gráficos.

4. Como desativo o plug-in Java da Sun?

Da janela do Internet Explorer, vá para a barra de opções e selecione Ferramentas > Opções da Internet. Quando aparecer a janela, selecione a guia Avançadas e efetue rolagem até ver a seção Java (Sun). Neste local você pode desmarcar a caixa de seleção para desativar o plug-in Java da Sun.

5. Quais navegadores são compatíveis?

O FunOrb oferece suporte a vários navegadores. Para obter a melhor experiência ao jogar, recomendamos um dos seguintes:

Microsoft Internet Explorer versão 6 ou superior.

Mozilla Firefox versão 1.5 ou superior.

Apple Safari.

6. O jogo está muito lento. O que está errado?

Se esta é a primeira vez que você joga um jogo específico do FunOrb, ou se tivermos atualizado o jogo desde a última vez que você jogou, ele pode demorar mais do que o normal para ser carregado, porque não haverá dados do jogo em cache no computador. Se esse não for o problema, considere o seguinte:

Um jogo lento normalmente é causado ou por jogar no FunOrb em um computador lento ou por uma conexão de baixa qualidade com a Internet entre o computador e nosso servidor. Mesmo que esteja usando um computador super-rápido conectado com um link ultrarápido, ainda assim é possível que uma das máquinas na rota entre nós e você não esteja funcionando adequadamente, resultando em um desempenho ruim ocasionalmente.

Para reduzir o problema, você pode:

  • 1. Tentar jogar em um outro servidor. A lista de servidores de jogos encontra-se nas páginas 'Informações do jogo', logo acima do botão 'Descrição'.

  • 2. Certificar-se de que não existam outros programas usando a conexão com a Internet. Feche quaisquer programas desnecessários que possam estar em execução no computador. Em especial, não tente jogar no FunOrb enquanto estiver fazendo download de arquivos ou vendo TV na web, porque não funcionará adequadamente!

  • 3. Tentar jogar em outras horas do dia, quando a Internet estiver menos congestionada e os vários roteadores não estiverem sobrecarregados.

  • 4. Na caixa de seleção "'Jogos" do site do FunOrb, há um link para "Opções Java". Aqui você verá uma caixa de seleção: clique no menu e selecione Miniaplicativo assinado usando Java da Sun e depois tente carregar o jogo da forma usual.

  • 5. Se estiver usando uma versão antiga do Windows, verifique se o compilador Java Justi-In-Time da máquina virtual está ativado. No Internet Explorer, em: Ferramentas > Opções da Internet > Avançadas > Microsoft VM.

  • 6. Tentar atualizar seu computador.

  • 7. Tentar usar outro provedor de acesso à Internet (ISP).


Nossos servidores têm uma conexão muito boa com a Internet, então é pouco provável que haja um problema de nosso lado.
