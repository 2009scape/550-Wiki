1. Comment fonctionne le mode de paiement PayBySMS ?

Vous envoyez un SMS au service PayBySMS et recevez un SMS contenant un code confidentiel à saisir sur notre site Web et qui vous fournira un mois d'abonnement. Le coût du SMS sera ajouté à votre facture téléphonique si vous disposez d'un contrat ou retiré du crédit de votre téléphone. Notez que votre opérateur peut également vous facturer le SMS envoyé.

2. Combien coûte l'abonnement avec ce mode de paiement ?

Le coût de l'abonnement varie en fonction du pays dans lequel vous résidez et des frais facturés par votre opérateur.

4. Dans quels pays ce mode de paiement est-il disponible ?

À l'heure actuelle, le service PayBySMS est disponible au Royaume-Uni, États-Unis, Australie, Autriche, Belgique, Canada, France, Danemark, Estonie, Finlande, Allemagne, Irlande, Lituanie, Pays-Bas, Norvège, Singapour et Suède.

5. Au bout de combien de temps mon abonnement sera-t-il actif ?

Vous devrez attendre 15 minutes une fois le sms reçu avant de pouvoir saisir votre code confidentiel. Une fois le code confidentiel saisi, le crédit d'abonnement sera ajouté à votre compte.

Si le code confidentiel que vous avez reçu et saisi ne fonctionne pas, contactez le service de facturation avant d'en acheter un autre.

6. Quel est le numéro auquel envoyer le sms ?

Le numéro auquel envoyer le sms vous est fourni lors du processus d'abonnement.

7. Comment puis-je effectuer un paiement ?

Il vous suffit de débuter un nouvel abonnement en sélectionnant S'inscrire à FunOrb sur notre site Web et de suivre les invites pour utiliser le mode de paiement PayBySMS. Veillez à suivre les instructions et à comprendre tous les points.

8. Comment résilier mon abonnement ?

Le paiement est uniquement effectué lorsque vous envoyez un sms au service PayBySMS et que l'on vous envoie un code confidentiel. Si vous ne souhaitez pas renouveler l'abonnement sur votre compte, il vous suffit de ne pas vous réabonner.

9. Combien de mois d'abonnement peut-on acheter à la fois ?

Vous pouvez acheter plus d'un mois de crédit à la fois. Cependant, vous pouvez uniquement ajouter du crédit d'abonnement PayBySMS à trois comptes à tout moment donné. Par conséquent, nous vous recommandons de ne pas dépasser cette limite sous peine de ne pouvoir utiliser le crédit supplémentaire acheté.

10. Combien de paiements seront-ils effectués ?

Un seul paiement est effectué à chaque fois que vous utilisez le service PayBySMS.

11. Le service PayBySMS n'est pas disponible dans mon pays. Que puis-je faire ?

Nous cherchons en permanence à proposer de nouveaux modes de paiement à nos joueurs. Si PayBySMS n'est pas encore disponible dans votre pays, ne nous soumettez pas de question à ce sujet. Veuillez plutôt consulter régulièrement les pages sur la facturation pour en savoir plus sur les nouvelles possibilités de facturation. En attendant, vous pouvez envisager d'utiliser un autre mode de paiement.

12. Mon code confidentiel ne fonctionne pas. Que puis-je faire ?

Vérifiez d'abord une nouvelle fois que vous avez patienté 15 minutes et correctement saisi le code confidentiel. Si le code confidentiel ne fonctionne toujours pas, soumettez une question au service de facturation en indiquant le numéro de téléphone utilisé pour le paiement, le code confidentiel que l'on vous a envoyé ainsi que la date et l'heure de l'envoi du SMS.

13. PayBySMS est disponible dans mon pays mais le logo PayBySMS ne s'affiche pas lorsque j'essaie de m'abonner. Pourquoi ?

Vous pouvez uniquement acheter un crédit d'abonnement à l'aide de PayBySMS si votre compte dispose de moins de 14 jours de crédit.

Si vous souhaitez utiliser ce mode de paiement, vous devez attendre qu'il reste moins de 14 jours de crédit d'abonnement sur votre compte.

14. J'ai envoyé un sms au service mais n'ai pas reçu de code confidentiel.

Un code confidentiel est envoyé à chaque utilisateur qui envoie un sms au service PayBySMS. Nous procédons à une facturation inversée ; vous n'êtes donc facturé qu'une fois le sms envoyé. Si vous n'avez pas reçu de code confidentiel, vous ne serez pas facturé.

Les quatre raisons principales sont les suivantes :

  • Votre opérateur bloque les SMS à tarif majoré entrants ;
  • Votre téléphone portable ne dispose pas d'un crédit suffisant pour payer un mois entier d'abonnement ;
  • Votre boîte de réception de SMS était saturée et le SMS contenant le code confidentiel n'a pas pu être reçu ;
  • Votre opérateur n'a pas pu envoyer le code confidentiel en raison d'un bug du service.

15. À combien de comptes Jagex peut-on s'abonner à l'aide du même téléphone portable ?

Vous pouvez vous inscrire pour un maximum de trois comptes différents à la fois en utilisant la même ligne téléphonique. Jagex se réserve le droit de bloquer/suspendre le crédit d'un compte, sans remboursement, si vous essayez de souscrire à plus d'abonnements Jagex sur une même ligne téléphonique que le nombre maximum autorisé.

16. Pourquoi n'ai-je reçu que la moitié de mon code ?

Vous n'avez peut-être reçu qu'une moitié de code pour l'une des raisons suivantes :

  • Votre opérateur n'a pas pu envoyer le code confidentiel en raison d'un bug du service ;
  • Le réseau téléphonique est surchargé ;
  • La boîte de réception de SMS de votre téléphone portable est pleine ;
  • Votre téléphone portable avait des problèmes de réception et le réseau n'a pas permis l'envoi de la totalité du code ;
  • Le crédit disponible sur votre téléphone portable n'était pas suffisant pour la réception des deux parties du code.

Si aucun des motifs ci-dessus ne correspond à votre situation et si vous n'avez pas reçu le reste du code dans les 72 heures, nous vous conseillons de contacter Surfpin.

17. Pourquoi ai-je reçu moins de crédits d'abonnement que prévu ?

Ce problème est généralement lié à l'envoi d'un SMS au numéro réservé aux abonnements à tarif préférentiel alors que votre compte ne bénéficie pas de ce tarif. Si vous tentez d'appliquer un code de tarif réduit à un compte plein tarif, vous paierez moins pour l'envoi du SMS mais vous recevrez moins de crédits d'abonnement (seulement 26 jours, en général).

Pour éviter les problèmes de ce type, vérifiez que vous envoyez votre SMS au numéro approprié. Veillez à effectuer la totalité du processus d'abonnement à chaque fois que vous vous abonnez. Vous pourrez ainsi vérifier que le numéro utilisé est correct.

Si vous êtes sûr d'avoir utilisé le bon numéro et si vous avez tout de même reçu moins de crédits d'abonnement que prévu, veuillez soumettre une question au service de facturation. Nous ferons les recherches nécessaires pour résoudre le problème.
