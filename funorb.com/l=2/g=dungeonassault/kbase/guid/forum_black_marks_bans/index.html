Infractions/exclusions des forums

  1. Les exclusions peuvent-elles être immédiates ?
  2. Comment se traduisent les sanctions de jeu sur les forums ?
  3. Je suis un abonné mais je n'arrive pas à publier des messages sur les forums. Pourquoi ?
  4. J'ai été exclu des forums. Pourquoi ? Que dois-je faire ?
  5. Pourquoi mon message a-t-il été supprimé du forum ?

1. Les exclusions peuvent-elles être immédiates ?

Si un message est jugé très offensant ou perturbateur, vous serez exclu immédiatement. Si vous répétez cette action plusieurs fois, vous recevrez un avertissement. Vous pouvez être exclu définitivement si l'infraction est jugée extrêmement offensante.

Si vous estimez que l'infraction n'est pas justifiée, vous pouvez faire appel en allant à la page Contester de la section Gestion du compte.

Pour en savoir plus :


Pour consulter le Code de conduite des forums, cliquez ici.

2. Comment se traduisent les sanctions de jeu sur les forums ?

Les infractions commises dans le jeu et les forums ont les mêmes conséquences et s'appliquent dans les deux cas. Un joueur sous le coup d'une sanction (exclusion ou blocage de discussion) pour infraction dans le jeu ne pourra pas poster sur les forums le temps de la sanction et inversement.

Pour en savoir plus :


Pour consulter le Code de conduite des forums, cliquez ici.
Pour en savoir plus sur les modérateurs de forum, cliquez ici.

3. Je suis un abonné mais je ne parviens pas à publier de messages sur les forums. Pourquoi ?

Si vous êtes un abonné, ou un non-abonné pouvant publier sur les forums, mais que vous ne pouvez pas créer de fil de discussion sur les forums ou répondre à celui de quelqu'un d'autre, il est possible que l'on vous ait exclu ou retiré temporairement le droit de publier des messages. La section précédente contient plus d'informations sur la manière dont les exclusions et les blocages de discussion affectent vos droits sur les forums.

Pour en savoir plus :


Pour consulter le Code de conduite des forums, cliquez ici.
Pour en savoir plus sur les modérateurs de forum, cliquez ici.

4. J'ai été exclu des forums. Pourquoi ? Que dois-je faire ?

Si vous êtes un abonné, ou un non-abonné pouvant publier sur les forums, vous avez dû enfreindre les règles des forums. Vous allez recevoir un message vous indiquant la nature et la date de l'infraction. Gardez à l'esprit que les forums sont également régis par les Règles de conduite et le Code de conduite des forums. Ils sont aussi régis par des règles spécifiques aux forums qui figurent dans un Post-it en haut de chaque forum. Pour contester un avertissement que l'on vous a adressé, vous pouvez utiliser le système de contestation. Pour ce faire, connectez-vous à votre compte et accédez à Compte > Gestion du compte > Contester.

Si vous n'êtes ni abonné, ni un non-abonné pouvant publier sur les forums, vous pourrez parcourir et effectuer des recherches sur les forums mais ne pourrez pas publier de message ni débuter des fils de discussion.

Pour en savoir plus :


Pour consulter le Code de conduite des forums, cliquez ici.
Pour en savoir plus sur les modérateurs de forum, cliquez ici.

5. Pourquoi mon message a-t-il été supprimé du forum ?

La décision de supprimer d'un forum un message ou un fil de discussion appartient aux modérateurs. Vous êtes donc prié de ne pas créer de fil de discussion pour demander le motif de la suppression d'un fil de discussion spécifique.

Si l'un de vos messages a été modifié ou supprimé, vous avez dû enfreindre l'une des règles du forum principal ou une règle spécifique à un forum.

Si votre message figurait simplement dans un fil de discussion qui a été supprimé, vous ne subirez aucune conséquence.

Si vous recevez un avertissement ou si votre message a été masqué, veuillez relire les Règles de conduite et le Code de conduite des forums pour être sûr que cela ne se reproduise pas.

Veillez à ce que vos messages soient corrects et respectueux. Nous ne tolérerons aucune campagne d'insultes. Les plaisanteries sont acceptables mais le fait d'injurier pour s'amuser ne sera pas toléré.

Nous ne tolérerons aucun type de harcèlement.

Il convient d'échanger des idées sans s'insulter. Ne vous éloignez pas du sujet de discussion et N'INSULTEZ PAS les autres utilisateurs.

Veillez à ce que vos messages soient décents. Ayez à l'esprit que nos forums accueillent des personnes de sexe masculin et féminin de diverses tranches d'âge, religions, nationalités, orientations sexuelles, etc. Pensez-y bien avant d'appuyer sur le bouton Publier.
