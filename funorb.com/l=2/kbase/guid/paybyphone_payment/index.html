1. Comment fonctionne le mode de paiement PayByPhone ?

Vous appelez notre service PayByPhone et recevez un code confidentiel à saisir sur notre site Web, qui vous fournira un mois d'abonnement. Le coût de l'appel sera ajouté à votre facture téléphonique.

2. Dans quels pays ce mode de paiement est-il disponible ?

Le service PayByPhone est actuellement disponible en Australie, Belgique, France, Allemagne, Suisse, Nouvelle-Zélande, aux Pays-Bas, États-Unis, au Royaume-Uni et au Canada.

3. Combien coûte l'abonnement avec ce mode de paiement ?

Le coût de l'abonnement varie en fonction du pays depuis lequel vous appelez et s'affiche lors du processus d'abonnement.

4. Au bout de combien de temps mon abonnement sera-t-il actif ?

Vous devrez attendre 15 minutes une fois l'appel passé avant de pouvoir saisir votre code confidentiel. Une fois le code confidentiel saisi, le crédit d'abonnement sera ajouté à votre compte.

Si vous n'arrivez pas à saisir votre code confidentiel pour une raison quelconque, contactez le service de facturation.

5. Comment puis-je effectuer un paiement ?

Sélectionnez S'abonner/Renouveler un abonnement sur www.funorb.fr et suivez les invites du mode de paiement par PayByPhone. Veillez à suivre toutes les instructions et à comprendre tous les points.

6. Comment résilier mon abonnement ?

Le paiement est uniquement effectué lorsque vous appelez le service PayByPhone et que l'on vous donne un code confidentiel. Si vous ne souhaitez pas renouveler l'abonnement sur votre compte, arrêtez simplement de faire des paiements.

7. Combien de mois d'abonnement peut-on acheter à la fois ?

Vous ne pouvez acheter qu'un seul mois de crédit d'abonnement à la fois.

8. À combien de comptes Jagex peut-on s'abonner à l'aide de la même ligne téléphonique ?

Vous pouvez vous inscrire pour un maximum de trois comptes différents à la fois en utilisant la même ligne téléphonique. Jagex se réserve le droit de bloquer/suspendre le crédit d'un compte, sans remboursement, si vous essayez de souscrire à plus de trois abonnements Jagex sur une même ligne téléphonique.

9. Peut-on appeler le service avec un téléphone portable ?

Il est possible d'utiliser le service PayByPhone avec un téléphone portable ; cependant, vous devez vous assurer que votre réseau vous autorise à appeler les services à tarif majoré. De plus, vérifiez que vous possédez un crédit suffisant sur votre téléphone pour l'appel au service. Notez que votre opérateur peut vous facturer des frais supplémentaires. Nous vous conseillons de vous renseigner auprès de votre opérateur à ce sujet.

10. Mon code confidentiel ne fonctionne pas. Que puis-je faire ?

Vérifiez d'abord une nouvelle fois que vous avez patienté 15 minutes et correctement saisi le code confidentiel.

Votre code peut ne pas fonctionner pour l'une des raisons suivantes :

  • Vous tentez d'utiliser un code dans un pays différent de celui de l'achat du code ;
  • Il y a au moins trois codes antérieurs non utilisés correspondant au numéro de téléphone que vous utilisez ;
  • Plus de trois comptes dont le mode de paiement est PayByPhone utilisent le même numéro de téléphone que vous.

Si le code confidentiel ne fonctionne toujours pas, soumettez une question au service de facturation en indiquant le numéro de téléphone utilisé pour le paiement, le code confidentiel que l'on vous a donné ainsi que la date et l'heure de l'appel

11. Quel est le numéro de téléphone du service ?

Le numéro de téléphone varie en fonction du pays depuis lequel vous appelez et vous sera fourni lors du processus d'abonnement.

12. Lorsque j'appelle le service, la communication ne s'établit pas correctement. Pourquoi ?

Renseignez-vous auprès de votre opérateur téléphonique sur votre ligne. Elle est peut-être bloquée pour les appels aux numéros à tarif majoré.

Les motifs possibles sont les suivants :

  • Le souscripteur de la ligne a bloqué les numéros à tarif majoré ;
  • Un appel vers un numéro à tarif majoré a fait l'objet d'un litige ;
  • Vous utilisez une ligne professionnelle.

Si votre ligne bloque les appels à tarif majoré, consultez la personne qui paie la facture téléphonique avant de demander la levée du blocage. Notez que vous ne serez pas facturé pour les appels qui ont échoué.

13. Ma ligne téléphonique ne bloque pas les numéros à tarif majoré mais je n'arrive pas à me connecter au service.

Nous n'empêchons aucun numéro d'appeler notre service PayByPhone. Si vous n'arrivez pas à joindre le service, veuillez contacter votre opérateur téléphonique. Vous pouvez également envisager d'utiliser un autre mode de paiement. Pour consulter la liste complète des modes de paiement, sélectionnez S'abonner/Renouveler un abonnement sur www.funorb.fr.

14. Le service PayByPhone n'est pas disponible dans mon pays. Que puis-je faire ?

Nous cherchons en permanence à proposer de nouveaux modes de paiement à nos joueurs. Si PayByPhone n'est pas encore disponible dans votre pays, ne nous soumettez pas de question à ce sujet. Veuillez plutôt consulter régulièrement les pages sur la facturation pour en savoir plus sur les nouvelles possibilités de facturation. En attendant, vous pouvez envisager d'utiliser un autre mode de paiement. Pour consulter la liste complète des modes de paiement, sélectionnez S'abonner/Renouveler un abonnement sur www.funorb.fr.

15. PayByPhone est disponible dans mon pays mais le logo PayByPhone ne s'affiche pas lorsque j'essaie de m'abonner. Pourquoi ?

Vous pouvez uniquement acheter un crédit d'abonnement à l'aide de PayBySMS si votre compte dispose de moins de 14 jours de crédit. Si vous souhaitez utiliser ce mode de paiement, vous devez attendre qu'il reste moins de 14 jours de crédit d'abonnement sur votre compte.

16. Pourquoi ai-je reçu moins de crédits d'abonnement que prévu ?

Ce problème est généralement lié à l'utilisation d'une ligne téléphonique d'abonnement à tarif préférentiel alors que votre compte ne bénéficie pas de ce tarif. Si vous tentez d'appliquer un code de tarif réduit à un compte plein tarif, vous paierez moins pour l'appel mais vous recevrez moins de crédits d'abonnement (seulement 26 jours, en général).

Pour éviter les problèmes de ce type, vérifiez que vous appelez le numéro approprié. Veillez à effectuer la totalité du processus d'abonnement à chaque fois que vous vous abonnez. Vous pourrez ainsi vérifier que le numéro de téléphone utilisé est correct.

Si vous êtes sûr d'avoir appelé le bon numéro et si vous avez tout de même reçu moins de crédits d'abonnement que prévu, veuillez soumettre une question au service de facturation. Nous ferons les recherches nécessaires pour résoudre le problème.
