Problèmes dans le jeu

  1. Les discussions ne s'affichent pas.
  2. Lorsque j'essaye de me connecter, on m'indique que mon nom d'utilisateur utilise déjà le service.
  3. Lorsque j'essaye de me connecter, le message « Connexion au serveur impossible » s'affiche.
  4. Le jeu se bloque dès que je consulte certaines cartes ou zones du jeu.
  5. Je rencontre des problèmes de son dans les jeux FunOrb.

1. Les discussions ne s'affichent pas.

Vérifiez que vous n'avez pas accidentellement modifié les options de configuration de jeu de sorte que les messages ne s'affichent pas dans la fenêtre de discussion. Par exemple, les discussions privée et publique sont peut-être désactivées. Elles peuvent être réactivées à partir de la partie inférieure de la fenêtre de jeu.

2. Lorsque j'essaye de me connecter, on m'indique que mon nom d'utilisateur utilise déjà le service.

De deux choses, l'une :

  • soit vous avez fini de jouer il y a quelques minutes et ne vous êtes pas déconnecté correctement. Il est donc possible que votre nom d'utilisateur soit toujours connecté au serveur. Vous serez déconnecté automatiquement au bout de quelques instants. Patientez donc quelques minutes, puis réessayez. Pensez à toujours vous déconnecter à l'aide de bouton de déconnexion car le fait de fermer un jeu FunOrb par un quelconque autre moyen peut vous faire perdre votre position actuelle dans le jeu.

  • soit les serveurs fonctionnent correctement pour les autres utilisateurs et vous n'arrivez toujours pas à vous connecter après avoir patienté un moment. Il est possible que quelqu'un se soit emparé de votre mot de passe et utilise votre compte. Pour savoir comment reprendre possession de votre compte, cliquez ici.

3. Lorsque j'essaye de me connecter, le message « Connexion au serveur impossible » s'affiche.

Essayez de vous connecter à un autre serveur. La liste des serveurs de chaque jeu se trouve à la page d'informations, juste au-dessus du bouton Description. Si l'un de nos serveurs connaît des problèmes techniques, cela résoudra le problème jusqu'à ce que nous le réparions.

Si le problème persiste :
Il existe peut-être un pare-feu sur votre réseau qui bloque la connexion. Contactez votre administrateur réseau ou votre fournisseur d'accès à Internet afin d'ouvrir les ports 43594 et 43595 (pour le trafic TCP entrant et sortant) qui sont utilisés par nos jeux. Nous ne sommes pas en mesure de résoudre ce problème car il est lié aux paramètres de votre connexion et non à une défaillance de nos jeux.

Si le problème persiste :
Il est possible que nos serveurs soient temporairement hors ligne à des fins de maintenance ou de mise à jour. Patientez une heure et réessayez.

4. Le jeu se bloque dès que je consulte certaines cartes ou zones du jeu.

Si votre ordinateur dispose de plus de 2 Go de mémoire (RAM) et si vous utilisez Sun Java, il se peut que Sun Java ne détecte pas correctement la quantité de mémoire disponible sur votre ordinateur.

Si cela ne s'applique pas à votre ordinateur, essayez de jouer en basse résolution pour voir si cela règle le problème pour la zone en question. S'il existe une zone particulière qui cause constamment un problème, envoyez un rapport de bug pour nous le signaler.

Pour résoudre le problème de détection incorrecte de la mémoire abordé ci-dessus, réalisez les opérations suivantes :

  • Dans le menu Démarrer, cliquez sur Paramètres > Panneau de configuration.

  • Dans le Panneau de configuration, double cliquez sur Java. La fenêtre Panneau de configuration Java s'ouvre.

  • Cliquez sur l'onglet Java puis sous Paramètres de l'applet Java Runtime, cliquez sur le bouton Afficher.

  • Dans la zone des paramètres d'exécution Java Runtime, saisissez -Xmx256M.

  • Appuyez sur OK pour fermer les fenêtres que vous avez ouvertes.

  • Redémarrez votre ordinateur pour appliquer les modifications.

5. Je rencontre des problèmes de son dans les jeux FunOrb.

Si vous rencontrez des problèmes de son dans un jeu FunOrb, vérifiez que votre carte son dispose des derniers pilotes. Pour ce faire, consultez le site Web du fabricant de votre carte son.
