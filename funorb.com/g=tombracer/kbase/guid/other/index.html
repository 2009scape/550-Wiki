1. How do I submit a technical query?

Before you send in a Technical Support form, please take a moment to make sure your computer is updated. You can do this by following the instructions below:

Keep Windows updated


To download upgrade patches for the Windows Operating System, please visit the Windows Download Center at: http://windowsupdate.microsoft.com/


Keep Java updated


The VM (Virtual Machine) used to run FunOrb games will need to be updated as new versions are developed and released. These updates are located on the Sun Java homepage, which can be found at the following address: java.com/en/download/index.jsp


Keep your web browser updated (e.g. Internet Explorer, Firefox)


Whichever web browser you use, we recommend that you use the latest version. This will help to run the game smoothly and help with your internet security.

Please update your web browser by visiting the relevant developer's official website, or by using the correct link that is supplied with the browser.


When your computer has been updated, please click HERE to continue.


2. How do I find my system information?

How to locate your system information for Windows Operating System


To locate your system software information, as well as some information regarding your hardware, perform the following steps:
  1. Start Menu > Settings > Control Panel
  2. Select the 'System' icon
  3. On the 'General' tab there should be information about what version of Windows you are running, how much RAM you have and, sometimes, the speed of your CPU

How to locate your Java version


To locate your Java software information (only if you have Sun Java installed):
  1. Start Menu > Settings > Control Panel
  2. Select the 'Java' icon
  3. Click the 'About' button to see the Java version you are running

How to locate your system information for Mac OS
  1. Click on the blue Apple icon in the top left corner of your screen and select 'About this Mac'.
  2. In the window that pops up, you will see the Operating System version as well as the processor and memory information for your Mac.

How to locate your browser's system information
  1. Open your browser (Internet Explorer, Firefox, etc.)
  2. Select the 'Help' menu and choose 'About'. This should display version information for your browser.

3. Can the FunOrb game window be resized?

If you are a subscribed member of FunOrb, then yes, there is a 'fullscreen' mode you can take advantage of. For non-members, however, there is no fullscreen mode.

Fullscreen mode requires the use of the signed applet. (The signed applet is the default. If you want to check or change this, please select 'Java Options' from the 'Games' menu at the top of this page.) We try to support fullscreen mode on as many computers as we can, but there will always be some computers where it doesn't work. If you have trouble, you could try upgrading your Java version or your graphics drivers.

4. How do I turn the SUN Java plug-in off?

From an Internet Explorer window, go to the option bar and select Tools > Internet Options. When this box appears select the Advanced tab and scroll down until you see the Java (Sun) section. Here you can uncheck the box to turn off the Sun Java plug in.

5. What browsers do you support?

FunOrb supports a variety of browsers. For the best gaming experience, we recommend one of the following:

Microsoft Internet Explorer version 6 onwards.

Mozilla Firefox version 1.5 onwards.

Apple Safari.

6. The game is running slowly, what is wrong?

If this is the first time you have played a particular FunOrb game, or if we have updated the game since the last time you played, please be aware that it may take longer to load than usual, as none of the game data will be cached on your computer. If this isn't the problem, then please consider the following:

A slow running game is normally caused by either playing FunOrb games from a slow computer or connecting on a poor quality Internet connection between your computer and our server. Even if you're using a super-fast computer connected using an ultra-fast cable link, it is still possible that one of the machines on the route from us to you isn't up to the task, resulting in poor performance occasionally.

To reduce the problem you can:

  • 1. Try playing on a different server. Game server lists can be found via the 'Game Info' pages - just above the 'Description' button.

  • 2. Make sure there aren't any other programs using your Internet connection. Close any unnecessary software which may be running on your computer. In particular, don't try playing FunOrb games at the same time as downloading files or watching Web TV as this doesn't work very well!

  • 3. Try playing at a different time of day, when the Internet is quieter and the various routers across the Internet aren't overloaded.

  • 4. In the 'Games' drop down menu on the FunOrb website is a link to 'Java Options. Here you will see a scroll with a drop down menu - Click on the drop down menu on this scroll and select Signed Applet using SUN Java and then proceed to load the game as normal.

  • 5. If you are using an older version of Windows, ensure your Java Just-In-Time compiler for virtual machine is enabled. In Internet Explorer, under: Tools > Internet-Options > Advanced > Microsoft VM.

  • 6. Try improving your computer.

  • 7. Try using a different ISP (Internet-Service-Provider).


Our servers have a very good Internet link so it's unlikely to be a problem at our end.
